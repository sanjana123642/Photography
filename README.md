# Photography
This project is a modern, single-page photography portfolio website designed to showcase visual work in a refined and elegant layout. It focuses on strong typography, immersive imagery, and a dark, cinematic theme suitable for creative professionals.

Overview

The website includes a full-screen hero section, an about section, service highlights, and a contact area. It is built using pure HTML and CSS, without frameworks, making it lightweight, fast, and easy to understand or customize.

Features

Full-screen hero section with background image and overlay

Minimal navigation bar with elegant typography

About section with image and personal introduction

Services grid for photography categories

Contact section with details and form layout

Dark theme with gold accent color

Clean, professional visual hierarchy

Technologies Used

HTML5

CSS3

Google Fonts (Playfair Display, Poppins)

Project Structure
index.html

All markup and styling are written in a single file for simplicity.

How to Run

Download or clone the project.

Open index.html in any modern web browser.

The website loads instantly without any setup.

Design Goals

Present photography work in a premium, distraction-free layout

Emphasize imagery through contrast and spacing

Maintain clean and readable code

Create a portfolio suitable for personal branding

Limitations

Static website only

No form submission handling

No image gallery or lightbox functionality

Possible Enhancements

Portfolio gallery with filters

Lightbox or modal image previews

Smooth scrolling and animations
